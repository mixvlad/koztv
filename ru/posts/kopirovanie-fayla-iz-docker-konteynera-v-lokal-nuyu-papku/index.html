Копирование файла из Docker-контейнера в локальную папку

2019-08-12

Проверьте containerId, выполнив эту команду:

sudo docker ps

Чтобы скопировать файл из контейнера на локальный компьютер, используйте следующую команду:

docker cp <containerId>:/путь_к_файлу_в_контейнере /локальный_путь

Полный путь к файлу внутри контейнера можно проверить этой командой:

readlink -f file.ext
