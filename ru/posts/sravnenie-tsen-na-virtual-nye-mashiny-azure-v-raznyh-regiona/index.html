Сравнение цен на виртуальные машины Azure в разных регионах

2017-06-20

Иногда не важно, где размещать виртуальную машину, поэтому нужно просто найти лучшую цену. Я предпочитаю использовать Microsoft Azure для своих проектов, так что могу размещать ВМ в разных местах. http://azureprice.net/

Там можно легко узнать, какой регион лучше для выбранного типа ВМ.
