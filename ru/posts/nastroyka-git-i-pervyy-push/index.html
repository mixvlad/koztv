Настройка Git и первый push

2019-05-04

Перейдите в локальную директорию проекта и создайте git-репозиторий:

git init

Чтобы добавить все файлы в следующий коммит, выполните:

git add .

Чтобы проверить статус git, используйте:

git status

Выполните команду для создания первого коммита:

git commit -m "Ваш комментарий"

Настройте ссылку на ваш github-репозиторий и отправьте изменения:

git remote add origin <Ссылка на GitHub Repo>     //связывает удалённый репо с локальным git repo

git remote -v                                  //проверка ссылки на удалённый репо

git push -u origin master                      // отправляет закоммиченные изменения в удалённый репо
