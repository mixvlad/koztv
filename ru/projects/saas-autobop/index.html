SaaS-система управления девелопментом для группы ПИК

2021-11-20

Комплексная SaaS-платформа для управления девелоперскими проектами и процессами в группе ПИК.

AutoBOP — система планирования и контроля строительных графиков. Мы создали приложение, которое решает проблемы девелоперов. Лендинг с подробностями. Это часть проекта Франшиза ПИК.

Любой проект — это набор событий, жёстко связанных друг с другом.

Это инструмент для планирования и контроля выполнения ключевых событий в процессе строительства.

Планируйте графики проектов, создавайте новые события, их связи и назначайте исполнителей.

Получает задачи, готовые к работе. Видит историю, кто какие действия выполнял. Планирует свои задачи по срокам и приоритетам. Может обсудить задачу со всеми заинтересованными сторонами прямо в карточке задачи.

Принимает исполнение, контролирует полноту и точность на основании подтверждающих документов.

Отслеживает рейтинги по исполнителям и нарушителям сроков, по процессам и проектам. Получает информацию о статусе проектов в разных представлениях: графические диаграммы, новостные ленты.

Скриншоты работающей системы.
