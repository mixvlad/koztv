---
title: "Как сгенерировать и скопировать SSH-ключ в Mac OS"
date: 2017-07-09
---

#### Генерация SSH-ключа

Для начала откройте окно терминала. Лично я использую [iTerm2](https://www.iterm2.com/).

Чтобы сгенерировать SSH-ключи, введите следующую команду в окне терминала:

```bash
ssh-keygen -t rsa
```

Вам нужно будет ответить на несколько вопросов в процессе генерации ssh-ключа. Оставьте расположение по умолчанию, на остальные вопросы отвечайте как хотите.

#### Копирование SSH-ключа в буфер обмена

Чтобы скопировать публичный ssh-ключ в буфер обмена, введите эту команду в окне терминала:

```bash
pbcopy < ~/.ssh/id_rsa.pub
```

#### Проверка парольной фразы SSH

Чтобы проверить парольную фразу SSH, введите:

```bash
ssh-keygen -y
```
